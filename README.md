# 🚀LangChain 大模型应用开发实战项目

---

## 「从入门到生产级」全栈学习代码仓库

> #### 本项目以**分阶段进阶 \+ 全可运行脚本**的形式，系统化覆盖大模型应用开发完整技术栈。基于 LangChain 核心框架，拆解 LLM 基础调用、提示词工程、结构化输出、RAG 检索增强、Agent 智能体五大核心模块，配套机试练习题，边学边练，轻松掌握大模型应用开发能力 🎯

---

## ✨ 项目核心亮点

|特性|详情说明|
|---|---|

## ⚙️ 快速上手指南

|步骤|操作说明|
|---|---|
|1️⃣ 环境准备|安装 Python 3\.10\+ 与 `uv` 包管理工具|
|2️⃣ 依赖安装|项目根目录执行 `uv sync`，一键安装全部依赖|
|3️⃣ 密钥配置|在 `.env` 文件中填入大模型 API 密钥与接口基础地址|
|4️⃣ 运行调试|进入对应章节文件夹，直接运行 Python 脚本即可学习调试|

## 🏗️ 整体技术架构

采用经典五层架构设计，自上而下解耦，兼顾学习可读性与生产级可扩展性：

|架构层级|核心技术栈|核心职责|
|---|---|---|
|基础设施层|Python 3\.x、uv、python\-dotenv|提供运行时环境、依赖管理、环境变量与密钥管理能力|
|模型服务层|兼容 OpenAI 协议大模型、Ollama、BGE\-M3 嵌入模型|提供文本生成、多模态理解、向量嵌入等核心模型能力|
|核心编排层|LangChain Core、LCEL 表达式、Pydantic、Runnable 体系|大模型应用编排核心，支持链式调用、结构化校验、并行执行、容错降级|
|功能组件层|提示词模板体系、输出解析器、文档加载器、文本切分器、向量数据库、工具调用框架|模块化封装 RAG、Agent 场景基础能力，可按需组合、灵活扩展|
|实战案例层|全章节示例代码、机试练习题|从入门到进阶的实战落地代码，覆盖大模型开发全栈知识点|
